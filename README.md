# Ocre
Observation et comparaison des résultats des élections

## Données
### Fonds de carte
Nous avons récupéré le shapefil des départements français sur le site IGN (). Nous les avons ensuite converti avec en geoJSON (WGS84) avec [mygeodata converter](http://converter.mygeodata.eu/), puis simplifiés avec [mapshaper](http://www.mapshaper.org/).
